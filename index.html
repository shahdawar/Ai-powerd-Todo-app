In-memory productivity workspace

Plan work clearly with a safer frontend-only todo app.

Create an account, sign in, add tasks, mark them complete, and switch between all, pending, and completed views. This app keeps data only in memory, so users, sessions, and tasks reset when the page reloads.

  • User signup, login, and logout
  • Admin and user roles
  • Add, complete, filter, and delete tasks
  • Responsive layout with smooth scrolling
